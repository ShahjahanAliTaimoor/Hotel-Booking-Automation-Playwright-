Hotel Booking App – Playwright Automation

This project contains end-to-end automated tests for a sample Hotel Booking web application using Playwright.
It covers major flows such as login, searching hotels, selecting rooms, filling booking details, and validating confirmation.

Features:
1. Playwright test framework
2. Page Object Model (POM)
3. Reusable utilities & selectors
4. Positive & negative test scenarios
5. Easy to run and extend

Tech Stack:
1. Node.js
2. Playwright
3. JavaScript / TypeScript (whichever you used)

How to Run:
1. npm install
2. npx playwright test

Project Structure:
1. tests/          → all test cases  
2. pages/          → page objects  
3. fixtures/       → test data  
4. playwright.config.js
